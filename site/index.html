Core Architecture

The Three Pillars

We bridge the gap between human developers, automated IDE tools, and agentic workflows to deliver error-free OTPless authentications.

01

The Developer CLI

A command-line companion utility (`otpless-cli`) built for developers. Perform automated project stack detection, generate custom playbooks, execute token/webhook live tests, and inspect error codes.

  • Commandsdetect · scaffold · verify · errors · live-test
  • EnvironmentNode.js / Terminal
02

The MCP Server

Exposes standard Model Context Protocol (MCP) tool endpoints. Integrates directly into AI-powered code editors (Claude Code, Cursor, Zed, Windsurf) so your assistant can parse docs, scaffold setups, and run verifications natively.

  • Protocol Toolsdetect_stack · get_docs · scaffold_integration · ...
  • IntegrationsStdio / HTTP server mode
03

Agentic Skills

Pre-packaged custom markdown skill manuals (`SKILL.md`) that teach LLMs the exact verification routines, carrier requirements, iOS exception domains, and callback rules for OTPless.

  • Supported AgentsClaude Code, Codex CLI, Custom GPTs
  • InstructionsPlatform configs & callback audits

Setup

Configure in minutes

Install the CLI utility globally or register the MCP server block to activate AI agent tools in your IDE.

1. Install CLI & MCP globally

shell 
npm install -g otpless-cli

2. Add to your Editor's MCP Server Config

Claude Code · Cursor · VS Code · Continue · Cline · Windsurf · Zed 
{
  "mcpServers": {
    "otpless-docs": {
      "command": "npx",
      "args": ["-y", "-p", "otpless-cli", "otpless-cli-mcp"]
    }
  }
}
  • Claude Code~/.claude/mcp.json
  • Cursorsettings → MCP
  • VS Code Copilot Chat.vscode/mcp.json
  • Continue~/.continue/config.yaml
  • Windsurf~/.codeium/windsurf/mcp_config.json
  • Zed~/.config/zed/settings.json
  • Clinesettings → MCP servers

3. Configure Agentic Custom Skills

Teach LLMs exactly how to integrate and verify OTPless auth. Import the relevant custom skill definition into your agent workspace:

  • Claude Code Skillskills/claude/SKILL.md
  • Codex CLI Skillskills/codex/SKILL.md
