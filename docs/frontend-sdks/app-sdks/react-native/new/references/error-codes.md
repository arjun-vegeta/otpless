# Error Codes

In React Native, error codes align with the respective native SDKs:

- **For Android**: React Native follows the error codes defined by the [Android SDK](https://otpless.com/docs/frontend-sdks/app-sdks/android/new/references/error-codes).
- **For iOS**: React Native follows the error codes defined by the [iOS SDK](https://otpless.com/docs/frontend-sdks/app-sdks/ios/new/references/error-codes).

This means that when handling errors in React Native, you should refer to the platform-specific error codes based on whether the app is running on Android or iOS.
