# Whatsapp

## Overview

The WhatsApp Cloud API Connector allows you to use your own WhatsApp number to send authentication messages to your users via OTPLESS. This enhances your brand presence and increases user trust by using a dedicated number for messaging.

## Benefits

- **Custom Branding**: Represent your brand with your own WhatsApp number and profile.
- **Higher Conversion Rates**: Boost user trust and conversion rates by using a branded channel for authentication messages.
- **Custom Templates**: Create and manage custom message templates directly from the OTPLESS dashboard for your authentication purposes.

## Prerequisites

Before you start, ensure you have:

- A phone number that is not already connected to WhatsApp.
- Completed Facebook Business Verification.

## Activation Steps

To activate the WhatsApp Cloud API Connector, follow these steps:

1. **Log in to OTPLESS Dashboard**: Access your OTPLESS account.
2. **Navigate to Connectors**: From the main menu, go to the "Connectors" section.
3. **Activate WhatsApp Cloud API**: Choose the WhatsApp Cloud API Connector and click on Activate.
4. **Start Onboarding**:
   - Click on "Start Onboarding" and follow the provided instructions.
   - Refer to the [WhatsApp Cloud API Reference](https://developers.facebook.com/docs/whatsapp/cloud-api/reference) for detailed guidance.
5. **Connect Your Number**: Enter and verify the phone number you wish to connect in the above onboarding process, ensuring it is not already linked to WhatsApp.

## Configuration

After completing the onboarding, configure your WhatsApp number with the following details:

1. **Profile Picture**: Upload a profile picture for your WhatsApp account.
2. **Business Information**: Fill in the necessary details for your WhatsApp profile, such as:
   - Address
   - Category
   - About
   - Description

## Testing

Test the WhatsApp Cloud API Connector to ensure everything is set up correctly:

1. **Send a Test Message**: Go to the testing section of the WhatsApp Cloud API Connector in the OTPLESS dashboard.
2. **Verify Message Receipt**: Ensure the test message is received on the connected WhatsApp number.
3. **Check Profile Details**: Confirm that the message displays the correct profile picture and business information.

## Troubleshooting

If you encounter any issues with the WhatsApp Cloud API Connector, try the following:

- **Verification Problems**: Ensure your Facebook Business Verification is up to date and completed.
- **Connection Issues**: Verify that the phone number is not already linked to another WhatsApp account.
- **Message Delivery Failures**: Check your internet connection and ensure the WhatsApp number is active and able to receive messages.
- **Configuration Errors**: Double-check the information provided during configuration for accuracy.

For further support, contact OTPLESS support or refer to the [WhatsApp Cloud API documentation](https://developers.facebook.com/docs/whatsapp/cloud-api/reference).
